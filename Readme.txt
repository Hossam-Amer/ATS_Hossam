ATS system which filters all applicants CVs based on the job description and extract important insights from the CVS such as matching percentage, Missing keywords, projects related to job description..etc..

to run it:
Open cmd in this folder and run command : pip install -U -r requirements.txt
To run the application run command : streamlit run main.py
